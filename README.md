**JPen** is a Java library for accessing pen/digitizer tablets and pointing devices. The [main documentation page](http://jpen.sf.net) is a good starting point to learn about JPen.

To build, run:

`mvn clean compile process-classes`

To execute the demo, run:

`mvn exec:exec`

I was able to get this to run on Windows 11 as of April, 2026. I'm using a Wacom Cintiq Pro 24, and the Wacom drivers are installed. Pressure sensitivity and tilt detection work with the demo.
